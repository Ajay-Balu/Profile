CSS

*NOTE : Browser inserts some CSS styling by DEFAULT!!!

  • Types of CSS implementations
  • InLine - Top priority
  • Internal - Medium Priority
  • External - Low Priority
  • Syntax
  • TAG Selector - TagName { property : value;} [low priority]
  • Class Selector - .ClassName { property : value;} [Group of multiple elements]
  • ID Selector - #IdName { property : value;} [Unique ID, Single element/tag]

Font Sizing

  • pixel
  • percentage - better for screen rezing but also inherit from parent [100% = 16px]
  • em - better for screen rezing but also inherit from parent [1em = 16px = 100%]
  • rem - same as em but doesn't inherit from parent [best use in most cases]

Float and Clear

  • Float used to make texts wrap around an elemet
  • Don't use it for Positioning, use absolute and relative for better pratice
  • clear is somewhat anti-float, clears the wrapping is its around an element
